同业存单基金研究报告

发表于 2022-03-18 分类于 金工 阅读次数： Valine：
本文字数： 15k 阅读时长 ≈ 19 分钟
这篇是受机构委托写的一个小课题。主要介绍了近期新出的一类可以用于现金理财的同业存单基金。当前市场中有同业存单基金6只，均跟踪中证同业存单AAA指数。2021年12月我国首批6只同业存单指数基金获批，并发行上市。总募集金额152.38亿元。同业存单指数基金未来或作为新的低波动理财工具承接部分货币基金与现金管理类理财产品的投资需求，未来市场前景广阔。同业存单基金拥有高于货币基金的收益率，由于主要投资标的为银行间同业存单，其风险介于货币基金与债券基金之间，即能够为投资者提供高于货币基金的收益又能有效控制回撤，其波动性要显著低于债券基金。相较于现金管理类理财产品，同业存单基金采用市价法估值，更加符合当前监管要求与未来发展趋势。截止2022年2月25日，除已上市的6只产品外，仍有87只同业存单基金向证监会递交了发起申请等待批复，未来发展前景广阔。

一、 同业存单指数基金现状

(一) 同业存单指数基金定义

根据基金合同，同业存单指数基金是一种被动指数基金。主要采用抽样复制和动态最优化的方法，投资于标的指数中具有代表性和流动性的成份券和备选成份券，或选择非成份券作为替代，构造与标的指数风险收益特征相似的资产组合，以实现对标的指数的有效跟踪[1]。

(二) 同业存单指数基金规模与业绩

2021年11月23日，根据 证监会网站披露的消息，共有6家基金公司（富国基金、鹏华基金、南方基金、中航基金、华富基金、惠升基金）的6只同业存单AAA指数7天持有期证券指数基金获批。该6只基金于2021年12月3日开始募集，基金基本信息如下。

从发行规模来看，6只基金的募资规模上限均为100亿元，且最终发行规模均未达上限。除南方基金的同业存单指数基金发行规模最大外，其余基金公司的同类产品发信规模相对较小。

6只同业存单指数基金产品中，富国、鹏华、惠升基金的产品收益率领跑，中航、华福、南方基金的收益率稍低。其中中航基金的同类产品风控更好，基本无回撤。

(三) 同业存单基金购买客户构成

根据基金招募说明书，基金的发售对象为符合法律法规规定的可投资于证券投资基金的个人投资者 、机构投资者、合格境外投资者及法律法规或中国证监会允许购买证券投资基金的其他投资人。暂不向金融机构的自营账户销售[2]。截至22年3月11日，上述基金尚未公布具体持有人占比。

二、 竞品分析

通过整理证监会对现金管理类理财产品、货币基金和同业存单AAA指数基金的相关规定，我们可以发现三种投资产品分别在投资范围、组合期限、投资集中度、估值方式等方面存在较为显著的区别。下表为海通证券整理的三种投资产品各方面的具体区别。

可以看出同业存单基金与现金理财与货币基金主要存在以下方面的区别。

1. 同业存单基金的存单投资集中度更高。从投资范围上来说，三者都可对同业存单、短期债券、资产支持债券等金融产品进行投资，但同业存单基金主要的投资方向依旧是同业存单，且要求对同业存单的投资比例不低于基金资产的80%；投资于标的指数成份券和备选成份券的比例不得低于非现金基金资产的 80%。

2. 杠杆要求较为宽松但流动性要求更加严格。现金理财与货币基金要求每只现金管理类产品持有不低于该产品资产净值5%的现金、国债、中央银行票据、政策性 金融债券。而同业存单基金要求持有不少于基金资产净值5%的现金或到期日在1年以内的政府债券。整体的流动性要求更加严格。其次现金理财的杠杆上限为120%，而同业存单基金的杠杆上限放宽至140%。

3. 同业存单基金久期更长。现金理财与货币基金规定久期不超过120天，平均剩余存续期限不得超过 240 天。而同业存单基金跟踪的同业存单AAA指数的修正久期为0.43，对应165天。

4. 估值方法不同。现金理财和货币基金主要采用的是摊余成本法进行会计核算。而同业存单基金主要采用市价法进行估值，更加符合监管的要求。

5. 投资者集中度与规模控制。同业存单基金要求单一投资者持有基金份额数不得达到或超过基金份额总数的50%、单日认购金额不超过1000 万元，并明确规定在募集期内规模上限为100亿元。于此对应的现金理财和货币基金主要是通过将规模与风险准备金挂钩，实现规模的控制，避免随意新发基金。

三、 市场前景与市场容量预判

通过第二部分的对现金管理类理财产品与货币基金的对比我们认为同业存单基金未来前景广阔，主要原因如下：

  1. 收益优势较为明显。通过测算我们发现截止2022 年3月11日，已经发行的6只同业存单基金的平均年化收益为2.68%（简单平均），其跟踪的同业存单AAA指数近3年化收益率为2.99%，对比中证货币基金指数近3年年化收益2.37%可以看出同业存单基金在收益率上存在较为显著的优势。

  2. 市价法估值符合未来发展方向。资管新规要求未来摊余成本法估值要逐步退出历史舞台，为了打破刚性兑付，市价法估值的金融产更加符合当前的监管要求和未来中国金融市场的发展方向。

  3. 各家基金公司积极排队布局。截止2022年3月11日，证监会官网显示，累计余100家基金公司争相申请布局同业存单指数基金，足见基金行业存单指数基金的重视。

目前已经发行的6只同业存单基金的募资规模不大，主要原因是在于同期有大量的新发基金，同业存单指数基金并没有重点产品去销售和宣传，也没有放在比较好的发行档期，只是作为比较普通的基金产品去发行。且其需要锁定7日的要求让投资者接受还需要一定时间。但是以其相较于货币基金与现金管理类理财产品的优势，同业存单基金很有可能成为未来许多家庭现金管理的升级版工具。

市场容量方面，我国目前同业存单存量在12.62万亿元左右，足够满足同业存单基金的配置需求。作为新兴的基金产品，我们预测同业存单基金由于存在收益优势，会承接一部分货币基金的投资需求；同时，由于资管新规对于现金类理财产品的各种限制要求，亦会吸引分流部分现金管理类理财产品的投资需求。

四、 同业存单指数基金成分存单的选择

作为被动指数基金，同业存单AAA指数基金的主要投资于标的指数的成分券以及备选成分券。但是为了更好的实现投资目标，同业存单AAA基金还可以投资于不属于成分券及备选成分券的其他同业存单、债券、短期融资券、中期票据等非金融企业业债务融资工具、资产支持证券、债券回购、银行存款、现金等货币市场工具等，以及法律法规或中国证监会允许基金投资的其他金融工具。

(一) 同业存单AAA指数编制方法介绍

根据《中证同业存单AAA指数编制方法》，中证同业存单 AAA 指数样本券由在银行间市场上市的主体评级为 AAA、发行期限 1 年及以下、上市时间 7 天及以上的同业存单组成。该指数以2014年12月31日为基日，以100为基点进行编制，采用市值集权计算，以反映信用评级为AAA的同业存单的整体表现。

中证同业存单AAA指数的成分券样本由满足以下条件债券构成：

a) 债券种类：在银行间市场上市的同业存单，债券币种为人民币；

b) 发行期限：1 年及以下；

c) 上市期限：7 天及以上；

d) 主体评级：AAA；

e) 付息方式：贴现式、或固定利率付息、或一次还本付息

中证同业存单 AAA 指数采用派许加权综合价格指数方法计算，计算公式为：
$$
\text { 报告期指数 }=\frac{\text { 报告期样本债券总市值 }+\text { 报告期样本券派息及在投资 }}{\text { 除数 }} \times 100
$$
其中，报告期样本债券总市值 $=\sum$ 全价 $\times$ 发行量 ，全价 $=$ 净价 $+$ 应计利息。中证同业存单 AAA 指数样本每个交易日自动调整一次，满足选样条件的债券当日起计入指数，不满足选样条件的债券次日起剔除出指数。

(二) 同业存单基金追踪指数方法介绍

被动指数基金跟踪指数目前主要存在两种方法：完全复制法与优化复制法。

完全复制法试图采用完全复制目标指数样本股的方法构建跟踪组合，即按照成分股在目标指数中的权重确定购买比例买入所有成分股，相对于其他复制方法而言，这种方法简单明了，构建的指数组合与目标指数之间保持高度一致，可以较好地实现指数的代表性和投资风险的分散性。但这种方法主要适合于成分股数量相对较少或成分股数量虽较多但流动性较好的指数。

优化复制法则是根据预先设定的标准剔除部分成分股并对其在组合中的相对权重进行优化再配置，从而使得构建出来的指数组合的跟踪成本及其与目标指数之间的跟踪误差控制在可以接受的范围内。优化复制法主要解决的问题是，投资人可以主动回避流动性较差、有财务或法律风险或者持有该股票受到法律限制的股票。此时，投资人可以采用具有代表性的替代策略，从指数成分股中选择与拟替代的成分股行业相近、定价特征类似、收益率相关性较高、具有代表性的个股来替代部分成分股。具体来讲，优化复制法又可以分为两种：

1）优化抽样法。主要通过对选定目标指数中各成分股的历史数据和相关指标进行分析，采用优化模型来寻找一个数量较小和误差范围可控的投资组合，使得该投资组合相对目标指数的跟踪误差最小。

2）分层抽样法。分层抽样法实际上是在优化抽样的基础上进一步添加了约束条件，也可以看作是两阶段优化方法，即第一阶段是确定大类权重，这一阶段十分重要，要将市场上所有证券按照特征分成20至30个行业（往往借鉴申万、中信的一级行业分类），以不同行业在目标指数中所占的权重来决定在该行业类别内的投资量；第二阶段是具体股票权重的优化再配置，利用最优化方法计算每个大类内股票的权重。

由于同业存单基金所追踪的成分券包含部分流动性不佳、被限制投资的同业存单，目前成立的6只同业存单AAA基金均采用优化复制法对目标指数进行跟踪。对于市场流动性不足、因法律法规原因个别成份券被限制投资等情况，通过投资其他成份券、非成份券等方式进行替代。
